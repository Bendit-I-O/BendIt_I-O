Class: Switch

Switch(swNum, socket, deviceNum)

An object that represents a switch output on a Device's associated Bendit board/circuit-bent device pair. An array of Switch objects are created when a new BenditDevice object is created.

Constructor

new Switch(swNum, socket, deviceNum)

Parameters:
Name Type Description
swNum number The switch channel (relay) of the Bendit board associated with this switch's device.
socket object The socket.io socket inhereted from the global Bendit-class instance. Do not change this.
deviceNum number The number of the Device associated with this switch. Passed on to be associated with the Bendit board this switch's device is assigned to.

Methods

setSwitch(v)

Sets the state of the switch on this device's Bendit board. Switches in the BenditDevice.switches array are zero indexed.
Parameters:
Name Type Description
v string | number The state of the switch. Accepts a string ("open"/"closed") or a number (0/1).

Example
//Create device and attach it to Bendit board 4
let toy = bendit.addDevice(bendit.addDevice({
"switches": 4,
"pots": 8,
"motors": 0,
"boardNumber": 4 
}); 
//Set state of Switch 4 channel on this Bendit board to "closed"
 toy.switches[3].setSwitch("closed"); //Switch 4 closes
  
 //Set state of Switch 4 channel to "open"
 toy.switches[3].setSwitch("open"); //Switch 4 is now open
